旋转吧大喵 使用说明
=======================

简介
----
这是一个专为《无限暖暖》设计的 Windows 窗口比例调整工具。通过快捷键让您能够轻松切换游戏窗口的横竖比例，完美满足各种构图需求。无论是竖构图拍摄、相册浏览还是留影沙漏，都能获得最佳的显示效果。

基本使用
--------
1. 【重要】请右键点击程序，选择"以管理员身份运行"
2. 程序启动后会在右下角显示托盘图标
3. 默认使用快捷选择模式，按下热键（默认 Ctrl+Alt+R）会弹出比例选择菜单




两种操作模式
-----------
1. 快捷选择模式（默认开启）
   - 按下热键后立即弹出比例选择菜单
   - 可以快速切换不同比例，随心旋转
   - 适合频繁切换不同构图的玩家

2. 常规模式
   - 按下热键自动调整到设定的比例
   - 再次按下热键会重置回原来的比例
   - 适合固定使用某个比例的玩家

窗口计算模式
-----------
1. 基于屏幕计算（默认开启）
   - 使用屏幕尺寸作为基准计算窗口大小
   - 可以获得最大的窗口尺寸
   - 适合全屏或大窗口游戏的玩家

2. 基于窗口计算
   - 使用窗口原始尺寸作为基准计算
   - 保持窗口的相对大小
   - 适合喜欢小窗口运行游戏的玩家

游戏窗口模式选择
--------------
1. 窗口模式（推荐）
   - 优点：支持所有比例的照片输出
   - 缺点：调整尺寸后可能需要在游戏内重新应用分辨率以修正边框

2. 全屏窗口模式
   - 优点：游戏体验更佳，支持垂直构图
   - 缺点：照片输出仅限显示器原生比例，更适合搭配常规模式

功能菜单
--------
右键点击托盘图标可以：
- 选择窗口：从列表中选择要旋转的窗口
- 窗口比例：选择预设比例或添加自定义比例
- 快捷选择模式：开启后热键直接弹出比例选择菜单
- 修改热键：设置新的快捷键组合
  * 提示：设置后可能有 1-2 秒延迟才会显示成功提示
- 显示提示：开启/关闭旋转成功的提示消息
- 窗口置顶：保持游戏窗口始终在最前
- 基于屏幕尺寸：切换窗口计算模式
- 退出：关闭程序

预设比例
--------
工具内置了多种常用比例：
- 横构图：32:9, 21:9, 16:9, 16:10, 5:4, 4:3, 3:2, 1:1
- 竖构图：2:3, 3:4, 4:5, 9:16（默认）

自定义比例
---------
1. 在托盘图标右键菜单中选择"添加自定义比例..."，会自动打开配置文件
2. 找到 [CustomRatio] 节，在 RatioList 后添加新的比例
   - 格式：用英文逗号分隔多个比例，每个比例用英文冒号连接宽高
   - 示例：16:10,17:10,18:10 表示添加三个比例
   - 支持任意合理的宽高比值
3. 保存配置文件后关闭
4. 重启软件即可在比例菜单中看到新添加的自定义比例

注意事项
--------
- 基于窗口计算时:
  * 程序会记录首次选择窗口时的原始尺寸，所有的尺寸调整都基于这个原始尺寸进行计算
  * 如果在游戏内重新应用了分辨率设置，需要在托盘菜单中重新选择目标窗口以更新尺寸记录
  * 重置功能会将窗口恢复到记录的原始尺寸

运行环境
--------
- Windows 10 或更高版本
- 需要安装 Visual C++ 运行库
  * 下载地址：https://aka.ms/vs/17/release/vc_redist.x64.exe
  * 如果运行时提示缺少 DLL，请安装此运行库
