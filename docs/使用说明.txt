旋转吧大喵 使用说明
=======================

简介
----
这是一个专为《无限暖暖》设计的 Windows 窗口旋转工具。只需一个快捷键，即可让游戏窗口优雅地旋转 90 度，完美解决竖构图的拍照问题。

基本使用
--------
1. 【重要】请右键点击程序，选择"以管理员身份运行"
2. 程序启动后会在右下角显示托盘图标
3. 按下快捷键（默认 Ctrl+Alt+R）即可旋转当前选择的窗口

功能菜单
--------
右键点击托盘图标可以：
- 选择窗口：从列表中选择要旋转的窗口
- 修改热键：设置新的快捷键组合
  * 提示：设置后可能有 1-2 秒延迟才会显示成功提示
- 显示提示：开启/关闭旋转成功的提示消息
- 自动隐藏任务栏：竖屏时自动隐藏任务栏（默认关闭）
- 退出：关闭程序

运行环境
--------
- Windows 10 或更高版本
- 需要安装 Visual C++ 运行库
  * 下载地址：https://aka.ms/vs/17/release/vc_redist.x64.exe
  * 如果运行时提示缺少 DLL，请安装此运行库

配置说明
--------
- 所有设置会自动保存存在程序目录下的 config.ini 文件中
- 包括：上次选择的窗口、快捷键设置、提示开关状态等
- 删除配置文件后会恢复默认设置
